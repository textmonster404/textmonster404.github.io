Greek Alphabet

The greek alphabet has been used to write the Greek language scince late 9th century or early 8th century BC. The greek alphabet is derived from the earlier Phoenician alphabet. It was the first language with distinct characters for each letter to have only certain letters be vowels, and others to be only consonants. This language is the ancestor of Latin and Cryllic scripts. It originally only had one case for each letter, but developed letter case at the same time as Latin; aquiring upper and lower cases for each letter.


Notes:
